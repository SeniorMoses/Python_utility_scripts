admission_system.py

SHS Admission Portal (CLI App)

A simple Python-based command-line application that simulates the Ghana SHS admission process using BECE grades.

 Features

- View available schools and programs
- Apply for admission
- Automatic admission decision based on cutoff grades
- Stores applicant data using JSON

Technologies Used

- Python
- JSON (for data storage)

How to Run

1. Clone the repository:
   git clone https://github.com/SeniorMoses/python_utility_scripts
2. Navigate into the folder:
   cd python_utility_scripts
3. Run the program:
   python admission_system.py
